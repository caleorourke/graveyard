# Blockquote Source

Wrap `small` around a name, date, title, or any other source within a `blockquote` element.

<blockquote>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
<small>Someone famous</small>
</blockquote>

```html
<blockquote>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>

```
